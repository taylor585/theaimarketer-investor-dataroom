Cost Explanation

Subscription allocation is a fixed estimate, not measured per-lead billing. In the current model, each run receives $1.00 from a $100/month subscription spread across 100 expected LeadHunter runs.
Effective subscription CPL is calculated after that: allocation divided by actual accepted leads. Roofing shows $1.00 / 14 accepted leads = $0.07 per lead.
OpenRouter AI cost is usage-based. Meta Ads is $14.46 total and Roofing is $4.12 total using the current GPT-5.5 assumption of $5/M input tokens and $30/M output tokens.
Elapsed time is shown only as a speed/efficiency signal, not as a cost line.
Electricians and Home Repairs show measured run cost in U.S. dollars. This uses all active run time at $35/hour plus the $1 subscription allocation and is kept separate from the selectable AI billing model.
